# JSC370-Final_Project

Website Link: https://patuanpurba.github.io/JSC370-Final_Project/

Title: What Makes a Movie Succeed? Predicting Rating and Revenue from Film and Team Characteristics
This project studies movie performance from 2000 to 2025 by analyzing how film characteristics, genres, and cast/crew experience relate to audience rating and revenue. It combines temporal trend analysis with predictive modeling to understand what factors are associated with movie success.

API-Used: TMDB API
- "https://api.themoviedb.org/3/discover/movie"
- "https://api.themoviedb.org/3/movie/{movie_id}"

Data Preparation:
- User can either use provided dataset or acquire new dataset with API Call through notebooks
- Run the notebook sequentially to get the same dataset structure as the one that is being used during analysis
- Data derivation is also located on the notebook. Therefore, user can use the last notebook to acquire "Fresh" Data Derivatives
